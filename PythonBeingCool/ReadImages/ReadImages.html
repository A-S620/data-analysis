Reading Images#

This code uses OpenCV and PyTesseract. It reads text from images.

The image being read: 

import cv2
import pytesseract

# Read image
path = "images/img.png"
img = cv2.imread(path)

text = pytesseract.image_to_string(img)
print(text)
